# 🛡️ Rede-Protege - Sistema Escolar de Denúncia de Bullying

Um projeto web pensado e elaborado para acolher e proteger vítimas de bullying no ambiente escolar. O sistema funciona como um canal de denúncias focado em manter o **anonimato absoluto** da vítima, garantindo a sua segurança e integridade.

## ⚖️ Base Legal
Este projeto foi fundamentado na **Lei 13.603/18**, que determina que serviços de disque-denúncia devem obrigatoriamente assegurar e proteger o sigilo das informações e a identidade do denunciante.

## ✨ Principais Funcionalidades

* **Acesso Restrito e Seguro:** Para evitar intrusões de pessoas externas, o sistema é acessível exclusivamente por alunos validados através do **ID de matrícula** escolar.
* **Denúncias Anônimas:** O fluxo de relato foi desenhado para facilitar que a vítima ou testemunha possa reportar a agressão sofrida sem expor sua identidade.
* **Protocolos Únicos:** Ao finalizar um relato, o sistema gera automaticamente um número de protocolo pseudoaleatório (baseado na data e horário exatos da submissão), permitindo o acompanhamento seguro.
* **Painel da Pedagogia:** Área restrita para a equipe pedagógica gerenciar, analisar e tomar providências sobre as denúncias recebidas.

## 🛠️ Tecnologias Utilizadas

* **Backend:** Python (Flask)
* **Banco de Dados:** MariaDB
* **Frontend:** HTML, CSS, JavaScript
